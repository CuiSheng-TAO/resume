# Brand Hero Design

**Date:** 2026-03-27

**Goal:** Replace the current generic landing hero with a stronger brand-led header: `Siamese Dream` as the primary name and `人人都有美观简历` as the softer right-side value line.

## Problem

The current hero feels off for two reasons:

- `人人都有美观简历` reads more like a slogan than a product name.
- `HR Companion Resume Studio` feels internal and generic, which weakens the emotional tone of the product.

## Approved Direction

The user approved:

- Use `Siamese Dream` as the long-term product name in the hero.
- Put `人人都有美观简历` on the right side as a softer supporting line.
- Demote the product promise (`3 分钟先起一版...`) so it no longer dominates the hero.

## UX Changes

- Left hero area:
  - remove `HR Companion Resume Studio`
  - make `Siamese Dream` the main headline
- Right hero area:
  - show `人人都有美观简历`
  - optionally keep the existing promise as smaller supporting copy
- Metadata:
  - update the page title to `Siamese Dream`

## Constraints

- Do not change the rest of the editing flow in this iteration.
- Keep the desktop hero balanced with the existing two-column shell.
- Preserve readability on mobile.

## Success Criteria

- The page reads as a branded product, not a generic demo.
- `Siamese Dream` is clearly the product name.
- `人人都有美观简历` remains visible, but as supporting copy rather than the product title.
